# LinkeDIT

## 1. Install

### 1.1. front-end


```sh
cd front-end
npm install
npm run start
```


### 1.2. back-end

#### database on docker, spring boot app on intellij IDEA
```sh
cd back-end/docker-sql
docker-compose up -d
```
shutdown with
```sh
docker-compose down
```

Start spring boot application, by running the main function
